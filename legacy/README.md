# Legacy SQL archive (non-authoritative)

Files in this folder are retained for historical reference and incident forensics only.
They are **not** used for deployment and must not be treated as the current source of truth.

## Archived paths
- `legacy/migrations/` (previous root-level SQL migrations)
- `legacy/supabase/schema.sql` (snapshot schema file)

## Authoritative path
- `supabase/migrations/` is the only authoritative migration source.
